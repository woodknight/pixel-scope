# PixelScope — Product Specification

## Overview
PixelScope is a cross-platform desktop image inspection tool for engineers.

## Core Principles
- Truth-first
- Instant feedback
- Minimal mental load
- Lean + extensible

## Core Features
- PNG/JPEG/TIFF support
- Infinite nearest-neighbor zoom
- Pixel inspector
- Histogram
- DNG + RAW support
- Binary RAW viewer

## Non-Goals
- Editing
- Cloud features
